# 跨域的问题

跨域的三种情况：

  1. 访问协议：http https
  2. 访问 ip: 192 172
  3. 访问端口：8301 9528

这三个出现任何一个不同都会出现跨域访问的问题。


跨域的本质就是浏览器对 ajax 请求的一种限制


解决：

  1. 在后端接口 Controller 上添加注解 @CrossOrigin// 跨域
  2. 使用 httpClient
  3. 通过 gateway 网关
